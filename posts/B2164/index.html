#Python

문제

Queue(큐)란?

  1. FIFO(First in First Out) 형식의 자료구조로 가장 먼저 들어온 데이터가 가장 먼저 나가는 형태의 자료구조이다. 파이썬에서는 collections.deque를 이용하면 쉽게 사용할 수 있다.

  2. 관련 용어는 아래와 같다.

enqueue: 데이터 삽입(맨 뒤에서 실행)
dequeue: 데이터 삭제(맨 앞에서 실행)
front: 큐의 맨 앞
rear: 큐의 맨 뒤
  1. 시간 복잡도는 아래와 같다.
enqueue: O(1)
dequeue: O(1)
search: O(N)
remove: O(N)
  1. 위의 문제는 큐 형식을 사용하면 쉽게 해결할 수 있다. dequeue를 활용하여 카드를 버리고, 다음 dequeue를 그대로 enqueue해준다면 다음 카드를 제일 아래로 옮기는 것과 같은 의미를 가진다.

제출 코드

import sys
from collections import deque

N = int(sys.stdin.readline().rstrip())
q = deque(range(1,N+1))
k=1
while len(q) >1:
    q.popleft()
    k = q.popleft()
    q.append(k)
print(k)
