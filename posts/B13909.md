## 제곱수 찾기 문제

![문제사진](/images/baek13909.png)

## 기존 답안
```python
import sys
N = int(sys.stdin.readline().rstrip())
    
L = [0] + [1]* (N)
for i in range(2, N+1):
    for j in range(i, N+1, i):
        if L[j]==0:
            L[j]=1
        else: L[j]=0
print(L.count(1))
```

1. 문제에 따라 for문을 사용하여 N까지 리스트 값에 따라 수정하도록 구현하였다. 주어진 메모리 제한이 64MB이기에 메모리 초과로 통과하지 못 한다.

2. 메모리 사용을 줄이기 위해 창문을 열고 닫음을 1과 0으로 표현한 걸 bool을 활용하여 구현하였다. 같은 방식이지만 int는 8byte를 사용하고 bool은 1byte를 사용하기에 약 8배에 해당하는 메모리 이득을 본다.

```python
import sys
N = int(sys.stdin.readline().rstrip())
L = [False] + [True] * N

for i in range(2, N+1):
    for j in range(i, N+1, i):
        L[j] = not L[j]

print(L.count(True))
```

3. 그러나 위 코드 역시도 메모리 초과가 되어 수학적 접근이 필요하다고 생각했고 실제로 해당 문제는 약수의 수가 홀수개인 N까지의 수를 찾는 문제이다. 이는 제곱수를 찾는 문제와 동일하므로 잘 알려진 제곱수의 개수인 루트N의 정수 부분을 출력하면 통과하게 된다.

```python
import sys
N = int(sys.stdin.readline().rstrip())
print(int(N**0.5))
```

4. 답안은 간단하지만 제곱수의 개념을 알고 있는지와 제곱수의 개수에 해당하는 식을 알고 있는지에 관한 문제이다.
