소수 찾기 문제

첫 제출 답안

import sys
N, M = map(int, sys.stdin.readline().rstrip().split())
for k in range(N, M+1):
    if k<2:
        continue
        
    d=1
    for p in range(2, int(k**(0.5))+1):
        if k % p == 0:
            d = 0
            break
    if d == 1:
        print(k)

  1. 기존의 코드는 N부터 M까지 각 수마다 for문을 돌아 소수인지 아닌지 판별하기에 비효율적이다.

  2. 위의 경우 특정 범위 내의 모든 소수를 한 번에 찾는 에라토스테네스의 체(sieve of Eratosthenes)를 활용하면 보다 효율적으로 구현할 수 있다.

  3. 이를 활용하여 M까지의 체를 만든 후 n~m 구간만 출력하면 원하는 문제의 답을 구할 수 있다.

수정 답안

import sys
N, M = map(int, sys.stdin.readline().rstrip().split())

def sieve(n, m):
  is_prime = [True] * (m + 1) # 모든 수를 소수로 가정
  is_prime[0] = is_prime[1] = False

  for i in range(2, int(m**0.5)+1):
    if is_prime[i]: # 소수라면? 2부터 시작.
      for j in range(i*i, m+1, i):
        is_prime[j] = False # i**2부터 i의 배수 제거
  
  primes = [i for i in range(n, m+1) if is_prime[i]]
  return primes # True인 i만 n~m까지 return

print(sieve(N, M))
